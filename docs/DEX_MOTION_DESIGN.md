# 도감 모션/그래픽 디자인 기획 (2026-07-19)

세계지도 탐험 캠페인의 리텐션 심장 = 도감. "세계가 **내 플레이로 잉크로 그려진다**"는
느낌을 모션으로 만든다. 아트 언어는 게임 그대로 — **종이(#F7F6F2) + 잉크(#23252E) +
파랑 액션(#2F6BFF)**. 컨셉 메타포: **탐험가의 지도책 / 필드 저널** — 빈 점(미지의 땅)이
잉크 실루엣(정복한 땅)으로 채워지고, 국가 완주 시 여권 스탬프가 찍힌다.

> 모션 토큰은 P4 이식분 재사용: fast120/normal240/slow400/celebration900,
> gentle(easeOutCubic)/spring(easeOutBack). 전 구간 OS 리듀스모션 대응(travel/overshoot
> 제거, 투명도만 유지). 잉크-드로우는 인게임 라인 렌더의 PathMetric 기술 재사용 = 저비용.

## 1. 수집 모먼트 — 도시 클리어 → 도감 편입 (핵심 payoff)

레벨 클리어 직후, 방금 푼 보드(화살표 빠진 도시 마스크)가 **도감 엔트리로 변신**하는 시퀀스:

| 단계 | 모션 | 타이밍 |
|---|---|---|
| ① 정착 | 남은 잉크 라인 사라지고 마스크 점만 남음 | 200ms |
| ② 잉크 번짐 | 중심에서 바깥으로 **잉크 블리드** — 셀이 점→잉크로, 스케일 0→1 easeOutBack 팝 (종이에 잉크 스미듯) | 900ms |
| ③ 스탬프 | "수집!" 배지가 회전+오버슈트로 쿵 + 잉크 스플래터 몇 방울 + 라이트 햅틱 | 320ms |
| ④ 정보 | 도시명 + 팩트 카드 아래에서 슬라이드업 | 240ms |

- 잉크 블리드 순서 = 중심 거리순(방사형)이 기본. 도시 실루엣이 "손으로 그려지듯" 나타남.
- 희귀/랜드마크 도시(수도·메가시티)는 스탬프에 파란 링 1회 펄스로 특별함 표시.

## 2. 세계지도 채우기 — 매크로 리빌

도감의 세계지도 화면: **잠긴 국가 = 흐린 점 군집**(우리 빈 보드 점과 동일 언어),
**수집 국가 = 채워진 잉크 실루엣**.
- 수집 후 지도로 돌아오면, 새로 딴 국가가 **점→잉크로 리빌**(위 ② 블리드 축소판, 400ms).
- 카메라가 그 국가로 부드럽게 팬/줌(gentle 400ms)한 뒤 채워짐 — "여기를 정복했다".
- 대륙별 진행바(잉크 게이지) + 세계 완성 % 상단 고정. 대륙이 채워질수록 지도가 짙어짐.

## 3. 국가 완주 — 비자 스탬프 (맵 위, 사용자 확정 2026-07-19)

⚠️ **별도 여권 화면 없음**(삭제). 스탬프는 **캠페인 맵의 국가지도 위**에 찍힌다.
- 조건: 그 국가의 **모든 도시 스테이지 + 국가 스테이지까지 전부 클리어**.
- 위치: **국가 맵 우상단 빈 공간**에 비자 스탬프(원형 이중링 + 지명 아치 + ★VISA★
  COLLECTED + 날짜, tools/gen_stamps.py). 회전하며 쿵(easeOutBack) + 성공 햅틱.
- ⚠️ **스테이지(보드) 중앙에는 절대 안 찍음** — 중앙 스탬프는 "재도전 불가"처럼 보임.
  도시 클리어의 수집 표시는 맵 핀의 초록 ✓로만.
- 미완주 국가 = 스탬프 없음. 라운드 헤더는 N/총(예: 5/5 ✓).

## 4. 마일스톤 — 대륙/세계 완성

- 대륙 완성: 그 대륙이 잉크로 "확" 채워지는 스윕 + 리본 배너(celebration 900ms, spring).
- 세계 100%: 지도 전체가 잉크로 완성되는 대형 드로우 시퀀스 + 특별 엔딩 카드.
- 절제 원칙: 축하는 마일스톤에만. 일반 수집은 조용하고 만족스럽게(과연출 금지).

## 5. 도감 브라우징 (정적이되 살아있게)

- 두 컬렉션 탭: **세계지도 모으기**(국가·도시·구) + **동물 도감**.
- 그리드: 수집=잉크 실루엣 카드 / 미수집=흐린 점 실루엣 위 "?" (미스터리로 완성 욕구 유발).
- 엔트리 탭 → 카드가 **페이지 넘김/플립**으로 열리며 실루엣 재드로우 + 정보(도시명·인구·
  국토 순위 등). 저널 페이지를 넘기는 느낌.
- 부분 수집 국가: 도시 N/M 진행 링 표시(예: 한국 3/5 도시).

## 6. 동물 도감 변주 — 표본 플레이트

동물은 지리가 아니므로 **자연사 박물관 표본 카드** 톤. 같은 잉크 실루엣 리빌이되,
카드 프레임을 표본 라벨(학명 자리·번호) 스타일로. 세계지도와 시각적으로 구분.

## 7. 구현 노트

- **잉크-드로우 = PathMetric 재사용**: 인게임 라인 애니와 동일 기술로 실루엣 외곽선 점진
  드로우 or 셀 방사형 리빌. CustomPainter 1장, GPU 부담 적음.
- **결정적**: 리빌 순서(중심거리)·스탬프 위치는 마스크에서 계산 → 플랫폼 무관 동일.
- **점→잉크 셀 보간**: 색 lerp(dot→ink) + 스케일 easeOutBack. 파티클 대신 셀 애니 = 가벼움.
- **여권/스탬프 에셋**: 잉크 스탬프 프레임은 gen_icon.py 계열로 절차 생성(스플래터 포함).
- 리듀스모션: 블리드/팝/스탬프 회전 제거, 크로스페이드로 대체.
